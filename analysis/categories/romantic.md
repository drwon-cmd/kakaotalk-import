# 로맨틱 관계 분석 가이드

## 개요

남녀 간(또는 이성/동성 간) 썸, 호감, 연애 가능성을 분석하는 가이드입니다.

---

## 1. 분석 프레임워크

### 관심/호감 스펙트럼

```
무관심 ---- 사교적 친절 ---- 호감 ---- 썸 ---- 연애 의향
   0           25              50       75        100
```

### 평가 영역

| 영역 | 평가 항목 |
|------|----------|
| **연락 패턴** | 빈도, 시작자, 응답 속도, 시간대 |
| **대화 내용** | 주제 깊이, 개인적 이야기, 미래 언급 |
| **언어 표현** | 호칭, 어미, 이모티콘, 특수 표현 |
| **관심 표현** | 질문 빈도, 기억력, 걱정/챙김 |
| **물리적 신호** | 만남 제안, 스킨십 언급, 선물 |

---

## 2. 호감 지표 상세

### 높은 호감 신호

**연락 패턴**:
- 먼저 연락하는 비율 50% 이상
- 응답 속도 빠름 (30분 이내)
- 심야/새벽 대화 자주
- 답장 길이가 김
- 대화 끝내기 아쉬워함

**대화 내용**:
- 개인적 이야기 자발적 공유
- 과거/트라우마/가족 이야기
- "우리", "같이" 표현 사용
- 미래 계획에 상대 포함
- 일상 사소한 것도 공유

**관심 표현**:
- 상대에 대한 질문 많음
- 이전 대화 내용 기억
- 중요한 일정 챙김
- 건강/안부 걱정
- 칭찬 자주

**물리적 신호**:
- 둘만의 만남 제안
- 선물/음식 챙김
- 외모 변화 알아챔
- 스킨십 관련 언급

### 낮은 호감/경계 신호

**연락 패턴**:
- 항상 상대가 먼저 연락
- 응답 지연 (몇 시간~하루 이상)
- 답장 짧고 건조함
- 대화 빨리 끝내려 함
- 읽고 답 안 함 자주

**대화 내용**:
- 업무/사무적 내용만
- 개인적 이야기 회피
- "우리" 대신 "나"/"너" 분리
- 미래 계획 언급 없음
- 다른 이성 언급 자주

**회피 신호**:
- 만남 제안 시 핑계
- "친구로 지내자" 뉘앙스
- 선 긋는 발언
- 연애 관련 질문 회피

---

## 3. 상황별 분석 포인트

### 썸 단계 확인

**Level 1: 관심 있음**
- 대화가 즐거움
- 상대 반응에 관심
- 아직 적극적 행동 없음

**Level 2: 썸 진입**
- 특별 대우 시작
- 둘만의 시간 만들기
- 은근한 신호 교환

**Level 3: 고백 직전**
- 명확한 호감 표현
- 만남 빈도 증가
- 관계 정의 필요성 인식

### 복잡한 상황들

**유부남/유부녀 케이스**:
- 숨기려는 시도 vs 오픈
- 배우자 언급 빈도/방식
- 미래 계획 현실성
- 패턴 반복 여부

**직장 내 관계**:
- 업무 vs 사적 대화 비율
- 근무 외 시간 연락
- 다른 동료와의 차별화
- 리스크 인식 수준

## 4. 상담 질문 흐름

### 기본 정보
1. 어떻게 알게 됐고 얼마나 됐나요?
2. 현재 상대의 연애/결혼 상태는?
3. 당신의 상태는?

### 신호 확인
4. 상대가 보내는 신호 중 가장 의미 있다고 생각하는 것?
5. 혹시 부정적/거리두는 신호도 있었나요?
6. 당신이 보낸 신호는?

### 심층 탐색
7. 이 사람에게 끌리는 진짜 이유?
8. 과거 연애 패턴과 비교했을 때?
9. 이 관계의 현실적 장애물?

### 욕구와 두려움
10. 가장 원하는 시나리오?
11. 가장 두려운 시나리오?
12. "알면서도 무시하는" 경고 신호가 있나요?

### 행동 계획
13. 지금까지 어떤 행동을 취했나요?
14. 왜 더 적극적으로 못 했나요?
15. 이 분석에서 원하는 것은?

## 6. 주의사항

### 분석가로서 주의할 점

1. **희망 편향 주의**: 의뢰인이 듣고 싶은 말만 하지 않기
2. **투사 주의**: 분석가 자신의 경험 투영 금지
3. **단정 금지**: "100% OO다"보다 "OO 가능성이 높다"
4. **윤리적 경계**: 불륜 조장/정당화 주의

### 의뢰인에게 전달할 메시지

- 분석은 참고 자료일 뿐
- 최종 판단과 행동은 본인의 몫
- 상대방도 변할 수 있음
- 직접 소통이 가장 정확함
