# 비즈니스 갈등 분석 가이드

## 개요

직장 동료, 사업 파트너, 상사-부하 관계에서 발생하는 숨겨진 갈등 요소를 분석하고 예방/해결 전략을 제안하는 가이드입니다.

---

## 1. 분석 프레임워크

### 갈등 단계 스펙트럼

```
잠재 갈등 ── 표면화 ── 확대 ── 고착화 ── 파국
    20          40       60       80       100
```

### 갈등 유형

| 유형 | 설명 | 예시 |
|------|------|------|
| **과업 갈등** | 업무 방식/결과 차이 | "이 방법이 더 나아" |
| **관계 갈등** | 개인적 감정 충돌 | "저 사람이 싫어" |
| **과정 갈등** | 역할/책임 분배 | "왜 내가 이걸 해야 해?" |
| **지위 갈등** | 권력/인정 경쟁 | "내가 더 인정받아야 해" |

---

## 2. 갈등 신호 탐지

### 초기 경고 신호 (Yellow Flags)

**언어적 변화**:
- [ ] 존댓말 ↔ 반말 갑작스러운 변화
- [ ] 호칭 격식화 ("OO씨" → "OO님")
- [ ] 이모티콘/감정 표현 감소
- [ ] 답장 길이 급격히 짧아짐

**대화 패턴 변화**:
- [ ] 응답 지연 증가
- [ ] 특정 주제 회피
- [ ] 업무 외 대화 감소
- [ ] CC 추가 (기록 남기려는 의도)

**톤 변화**:
- [ ] 냉소적 표현 증가
- [ ] 수동적 공격 ("네~ 알겠습니다~")
- [ ] 과도한 형식성
- [ ] 책임 회피 언어 ("제가 알기론...", "확인 필요...")

### 심각 경고 신호 (Red Flags)

**명시적 갈등**:
- [ ] 직접적 비난/비판
- [ ] 공개적 반박
- [ ] 무시/무응답
- [ ] 협조 거부

**조직적 징후**:
- [ ] 다른 동료에게 험담
- [ ] 상위 보고 (고자질)
- [ ] 증거 수집 행동
- [ ] 회의 불참/배제

## 3. 권력 역학 분석

### 권력 원천

| 원천 | 설명 | 대화에서 나타남 |
|------|------|----------------|
| **직위 권력** | 공식적 지위 | "결정권은 제가..." |
| **전문 권력** | 기술/지식 우위 | "이건 제가 더 잘 아는데..." |
| **정보 권력** | 정보 접근성 | "제가 들은 바로는..." |
| **관계 권력** | 네트워크/인맥 | "대표님께 말씀드리면..." |
| **보상 권력** | 인사/평가 영향 | "평가 때 반영..." |

### 권력 불균형 패턴

**상위자의 갈등 표현**:
- 무시, 묵살
- 공개적 질책
- 정보 차단
- 기회 배제

**하위자의 갈등 표현**:
- 수동적 저항
- 소극적 업무 태도
- 뒷담화, 불평
- 이직 준비

**동등 관계 갈등**:
- 경쟁심
- 공 가로채기
- 책임 떠넘기기
- 영역 다툼

## 4. 갈등 원인 분석

### 표면적 vs 실제 원인

| 표면적 원인 | 실제 원인 가능성 |
|------------|-----------------|
| "업무 방식이 안 맞아" | 인정받지 못한다는 느낌 |
| "커뮤니케이션 문제" | 신뢰 부족 |
| "역할이 불명확해" | 권력/영역 다툼 |
| "공정하지 않아" | 질투, 상대적 박탈감 |

### 숨겨진 갈등 촉발 요인

- **인정 욕구 좌절**: "내 기여가 무시당함"
- **영역 침범**: "내 일에 간섭함"
- **가치관 충돌**: "일하는 철학이 다름"
- **과거 앙금**: "예전에 당한 일"
- **외부 스트레스 전이**: "다른 문제를 여기서 풀음"

## 5. 상담 질문 흐름

### 관계 구조 파악
1. 조직 내 이 사람과의 관계는? (상사/부하/동료/파트너)
2. 업무적으로 서로 얼마나 의존하나요?
3. 이 사람 없이 업무 가능한가요?

### 갈등 히스토리
4. 언제부터 문제가 시작됐나요? 특정 사건?
5. 이전에 비슷한 갈등 있었나요?
6. 이 사람이 다른 동료와도 갈등이 있나요?

### 현재 상태
7. 지금 갈등 수준은? (잠재/표면화/확대/고착)
8. 이 갈등을 아는 사람은?
9. 상위 관리자의 입장은?

### 본인 역할 인식
10. 이 갈등에서 당신의 역할/기여는?
11. 상대가 당신에게 불만인 이유가 뭘까요?
12. 당신이 양보할 수 있는 것과 없는 것은?

### 해결 의지
13. 이 관계를 개선하고 싶은가요, 최소화하고 싶은가요?
14. 해결 안 되면 어떻게 되나요?
15. 최악의 시나리오와 최선의 시나리오는?

## 6. 대화 분석 체크리스트

### 갈등 심화 언어 패턴

```
[ ] 일반화: "항상", "절대", "매번"
[ ] 비난: "너 때문에", "네가 그러니까"
[ ] 경멸: 비꼬기, 냉소, 무시
[ ] 방어: 변명, 책임 회피
[ ] 담쌓기: 대화 거부, 무응답
```

### 건강한 갈등 해결 신호

```
[ ] "나"메시지 사용: "내가 느끼기에..."
[ ] 구체적 행동 언급: "그때 OO한 게..."
[ ] 해결 의지 표현: "어떻게 하면 좋을까요?"
[ ] 상대 관점 인정: "네 입장도 이해해"
[ ] 미래 지향: "앞으로는..."
```

## 8. 해결 전략 프레임워크

### Thomas-Kilmann 갈등 해결 모드

| 모드 | 적합한 상황 |
|------|------------|
| **경쟁** | 긴급, 중요한 이슈, 확실히 옳을 때 |
| **협력** | 양측 관심사 모두 중요, 시간 있을 때 |
| **타협** | 중간 정도 중요, 빠른 해결 필요 |
| **회피** | 사소한 이슈, 냉각기 필요 |
| **수용** | 상대가 옳을 때, 관계 유지가 더 중요 |

### 대화 기반 해결 단계

1. **냉각**: 감정적 대화 피하기
2. **사실 확인**: "제가 이해한 게 맞나요?"
3. **감정 인정**: "불편하셨을 것 같아요"
4. **관점 공유**: "제 입장은..."
5. **공동 해결**: "어떻게 하면 좋을까요?"
6. **합의 문서화**: 명확히 기록

## 9. 주의사항

### 분석가로서 주의할 점

- 한쪽 편 들지 않기
- 법적/HR 이슈는 전문가 연계
- 괴롭힘/차별 발견 시 보고 의무 안내
- 개인정보 보호

### 한계 인식

- 대화만으로는 전체 맥락 파악 어려움
- 상대방 관점 직접 들을 수 없음
- 조직 문화/맥락 고려 필요
