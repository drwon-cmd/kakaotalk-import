# 카카오톡 대화 분석 방법론

## 1. 분석 단계

### Phase 1: 초기 스캔 (Initial Scan)
- 대화 참여자 식별
- 대화 기간 및 빈도 파악
- 주요 토픽 추출
- 대화 톤 및 분위기 개요

### Phase 2: 패턴 분석 (Pattern Analysis)
- **시간 패턴**: 언제 대화하는가? (심야, 업무시간, 주말 등)
- **빈도 패턴**: 누가 먼저 연락하는가? 응답 속도는?
- **길이 패턴**: 누가 더 길게 쓰는가? 관심의 지표
- **이모티콘/스티커 패턴**: 감정 표현 방식

### Phase 3: 언어 분석 (Linguistic Analysis)
- **호칭 변화**: 형식적 → 친밀한 호칭 전환점
- **어미 변화**: 존댓말 ↔ 반말 전환
- **특수 표현**: 둘만의 언어, 내부 농담
- **감정 표현 빈도**: 기쁨, 걱정, 불만 등

### Phase 4: 관계 역학 분석 (Relationship Dynamics)
- **권력 균형**: 누가 주도권을 가지는가?
- **의존도**: 감정적/실용적 의존 관계
- **경계선**: 어디까지 공유하는가?
- **갈등 패턴**: 갈등 시 어떻게 반응하는가?

### Phase 5: 심층 해석 (Deep Interpretation)
- **숨겨진 의도**: 표면 아래의 진짜 메시지
- **방어기제**: 회피, 합리화, 투사 등
- **전이 패턴**: 과거 관계 패턴의 반복
- **미래 예측**: 현재 패턴이 지속될 경우

## 2. 핵심 분석 지표

### 관심/호감 지표 (Interest Indicators)
| 지표 | 높은 관심 | 낮은 관심 |
|------|----------|----------|
| 응답 속도 | 빠름 (분 단위) | 느림 (시간~일 단위) |
| 대화 시작 | 먼저 연락함 | 항상 받기만 |
| 메시지 길이 | 상세하고 길게 | 짧고 건조하게 |
| 질문 빈도 | 상대에 대해 자주 물음 | 질문 거의 없음 |
| 개인 정보 공유 | 자발적으로 많이 공유 | 최소한만 공유 |
| 미래 언급 | "다음에 같이..." | 미래 계획 언급 없음 |

### 갈등 신호 (Conflict Signals)
| 신호 | 설명 |
|------|------|
| 응답 지연 증가 | 갑자기 답이 느려짐 |
| 어미 변화 | 반말 → 존댓말 전환 |
| 이모티콘 감소 | 감정 표현 줄어듬 |
| 짧은 답변 | "ㅇㅇ", "ㅋ", "알겠어" |
| 화제 회피 | 특정 주제 피함 |
| 수동적 공격 | 빈정거림, 냉소적 표현 |

### 신뢰/친밀 지표 (Trust/Intimacy Indicators)
| 지표 | 설명 |
|------|------|
| 취약성 공유 | 약점, 고민, 두려움 공유 |
| 과거 트라우마 언급 | 아픈 기억 공유 |
| 미래 계획 포함 | "우리", "같이" 표현 |
| 일상 공유 | 사소한 일도 공유 |
| 조언 요청 | 중요한 결정에 의견 구함 |

## 3. 심리학적 프레임워크

### 애착 이론 (Attachment Theory)
- **안정 애착**: 균형잡힌 대화, 적절한 거리
- **불안 애착**: 과도한 연락, 확인 요구
- **회피 애착**: 감정 표현 회피, 거리두기
- **혼란 애착**: 불규칙한 패턴, 밀당

### 교류 분석 (Transactional Analysis)
- **부모 자아 (Parent)**: 훈계, 조언, 보호
- **성인 자아 (Adult)**: 논리적, 사실 기반
- **아이 자아 (Child)**: 감정적, 충동적, 장난

### 권력 역학 (Power Dynamics)
- **one-up**: 주도권 잡으려는 시도
- **one-down**: 복종, 맞추려는 시도
- **one-across**: 대등한 관계 유지

## 4. 분석 시 주의사항

### 해야 할 것
- 전체 맥락 고려
- 여러 가능성 열어두기
- 패턴 기반 해석
- 추가 정보 요청 (Q&A)

### 하지 말아야 할 것
- 단일 메시지로 결론 내리기
- 분석가의 편견 투영
- 과도한 일반화
- 확정적 예측

## 5. Q&A 세션 가이드

### 목적
대화 분석만으로 알 수 없는 맥락과 배경 정보를 상담식 질문을 통해 수집

### 원칙
1. **한 번에 하나씩** - 질문 폭격 금지
2. **열린 질문** - "예/아니오"보다 서술형
3. **판단 배제** - 중립적 톤 유지
4. **점진적 심화** - 표면 → 심층으로

### 질문 순서
1. 관계의 기본 정보
2. 과거 맥락 및 이력
3. 현재 감정 상태
4. 욕구와 두려움
5. 최종 목표 확인
