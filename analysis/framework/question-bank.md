# 상담식 질문 템플릿

## 공통 질문 (모든 카테고리)

### 1단계: 기본 정보 수집
```
Q1. 이 대화 상대와 처음 어떻게 알게 되셨나요? 얼마나 됐나요?

Q2. 현재 이 관계를 한 마디로 정의한다면?

Q3. 이 대화를 분석하고 싶은 특별한 이유가 있나요?
```

### 2단계: 맥락 파악
```
Q4. 대화에는 없지만 제가 알아야 할 배경 정보가 있나요?

Q5. 이 사람에 대해 다른 사람들은 어떻게 평가하나요?

Q6. 당신이 아는 이 사람의 과거 관계 패턴이 있나요?
```

### 3단계: 감정 탐색
```
Q7. 이 사람과 대화할 때 주로 어떤 감정이 드나요?

Q8. 이 관계에서 가장 불안하거나 불편한 점은?

Q9. 반대로 가장 좋거나 편안한 점은?
```

### 4단계: 욕구와 두려움
```
Q10. 이 관계에서 당신이 가장 원하는 것은?

Q11. 가장 두려운 시나리오는?

Q12. 최선의 시나리오는?
```

### 마무리: 목표 확인
```
Q13. 이 분석을 통해 얻고 싶은 것은 정확히 무엇인가요?
- 객관적 현실 직시?
- 구체적인 행동 제안?
- 관계 종료 결정?
- 그냥 누군가에게 털어놓고 싶은 것?
```

## 로맨틱 관계 분석 질문

### 상대방 파악
```
R1. 이 사람의 과거 연애/결혼 이력을 알고 있나요?

R2. 현재 이 사람이 만나는 다른 이성이 있나요?

R3. 이 사람이 당신에게 보내는 신호 중 가장 의미 있다고 생각하는 것은?
```

### 자기 인식
```
R4. 당신이 이 사람에게 끌리는 가장 큰 이유는?

R5. 이 사람이 당신에게 끌릴 만한 이유가 있다면?

R6. 당신의 과거 연애 패턴이 있다면? (항상 OO한 사람에게 끌린다 등)
```

### 현실 체크
```
R7. 이 관계가 실제로 발전한다면, 현실적인 장애물은?

R8. 주변 사람들(친구, 가족)은 이 관계를 어떻게 볼 것 같나요?

R9. 이 사람에 대해 "알면서도 무시하는" 경고 신호가 있나요?
```

### 행동 평가
```
R10. 지금까지 당신 쪽에서 어떤 행동을 취하셨나요?

R11. 아직 하지 못한 것 중 하고 싶은 것은?

R12. 왜 아직 못 했나요?
```

## 비즈니스 갈등 분석 질문

### 관계 구조
```
B1. 조직 내 이 사람과의 관계는? (상사/부하/동료/파트너 등)

B2. 업무적으로 서로 얼마나 의존하는 관계인가요?

B3. 이 갈등이 업무 성과에 미치는 영향은?
```

### 갈등 히스토리
```
B4. 언제부터 갈등이 시작됐나요? 특정 사건이 있었나요?

B5. 과거에 비슷한 갈등을 해결한 경험이 있나요?

B6. 이 사람이 다른 동료들과도 갈등이 있나요?
```

### 권력 역학
```
B7. 이 갈등에서 누가 더 "힘"이 있나요?

B8. 조직 내 누가 이 갈등을 알고 있나요?

B9. 상위 관리자/경영진은 어떤 입장일 것 같나요?
```

### 해결 의지
```
B10. 이 관계를 개선하고 싶으신 건가요, 아니면 최소화/종료하고 싶으신 건가요?

B11. 이 갈등이 해결되지 않으면 어떻게 되나요?

B12. 당신이 양보할 수 있는 것과 절대 양보할 수 없는 것은?
```

## 친구 관계 위험신호 분석 질문

### 관계 이력
```
F1. 이 친구와 얼마나 오래 알고 지냈나요?

F2. 예전과 지금, 관계가 어떻게 달라졌나요?

F3. 변화가 시작된 시점이나 계기가 있나요?
```

### 현재 상태
```
F4. 요즘 이 친구와 연락 빈도는 어떤가요?

F5. 최근 만났을 때의 분위기는 어땠나요?

F6. 이 친구가 다른 친구들과는 어떻게 지내나요?
```

### 감정 탐색
```
F7. 이 친구에게 서운한 점이 있다면?

F8. 혹시 이 친구가 당신에게 서운해할 만한 일이 있었나요?

F9. 이 관계에서 당신의 역할은 주로 무엇이었나요?
   (조언자? 들어주는 사람? 에너지 공급자? 등)
```

### 미래 전망
```
F10. 이 우정이 어떻게 되길 원하시나요?

F11. 이 관계가 끝난다면 얼마나 영향이 있을까요?

F12. 이 친구에게 직접 물어본 적 있나요? "우리 사이 괜찮아?"
```

## 상황별 후속 질문

### 상대방이 모호한 신호를 보낼 때
```
- 그 행동/말이 다른 사람에게도 하는 건가요, 당신에게만?
- 그 순간 당신은 어떻게 반응했나요?
- 그 후 분위기가 어떻게 변했나요?
```

### 분석 대상자가 방어적일 때
```
- 이 질문이 불편하시면 넘어가도 됩니다.
- 다른 방식으로 여쭤볼게요...
- 지금 느끼시는 감정을 말씀해주실 수 있나요?
```

### 핵심에 가까워졌을 때
```
- 지금 말씀하신 부분이 중요한 것 같은데, 조금 더 설명해주실 수 있나요?
- 그때 정확히 어떤 생각이 드셨나요?
- 그 감정을 한 단어로 표현한다면?
```

### 마무리할 때
```
- 오늘 이야기하면서 새롭게 깨달은 것이 있나요?
- 더 하고 싶은 말씀이 있으신가요?
- 분석 결과를 들을 준비가 되셨나요?
```
