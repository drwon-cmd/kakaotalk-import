# 심리학적 분석 모델

## 1. 애착 이론 (Attachment Theory)

### Bartholomew & Horowitz 4분류

| 유형 | 자기 인식 | 타인 인식 | 대화 특징 |
|------|----------|----------|----------|
| **안정형** | 긍정적 | 긍정적 | 균형잡힌 주고받기, 적절한 자기개방 |
| **불안형** | 부정적 | 긍정적 | 과도한 연락, 확인 요구, 읽씹 민감 |
| **회피형** | 긍정적 | 부정적 | 감정 표현 회피, 짧은 답변, 거리두기 |
| **혼란형** | 부정적 | 부정적 | 불규칙한 패턴, 밀당, 갑작스러운 변화 |

### 대화에서 애착 유형 탐지

**불안형 신호**:
- "왜 답 안 해?", "바빠?", "화났어?" 자주 물음
- 긴 메시지 연속 전송
- 상대 반응에 과민
- 확인받고 싶어하는 표현 다수

**회피형 신호**:
- 감정적 대화 회피
- 짧고 사무적인 답변
- 깊어지려 할 때 화제 전환
- "나 바빠", "나중에" 자주 사용

## 2. 교류 분석 (Transactional Analysis)

### 자아 상태 (Ego States)

| 자아 상태 | 대화 특징 | 예시 |
|----------|----------|------|
| **비판적 부모 (CP)** | 훈계, 지적, 평가 | "그렇게 하면 안 되지", "왜 그랬어?" |
| **양육적 부모 (NP)** | 위로, 보호, 지지 | "괜찮아", "걱정 마", "잘 될 거야" |
| **성인 (A)** | 논리적, 객관적, 정보 교환 | "그래서 결론은...", "팩트만 말하면..." |
| **자유로운 아이 (FC)** | 감정적, 충동적, 장난 | 이모티콘 다수, 장난, 흥분 표현 |
| **순응하는 아이 (AC)** | 맞춤, 복종, 눈치 | "알겠어", "네 말이 맞아", "미안해" |

### 교류 패턴 분석

**상보적 교류 (건강)**: A→A, NP→FC 등 예측 가능
**교차 교류 (갈등)**: A→A 기대했는데 CP→AC로 반응
**이면 교류 (숨은 의도)**: 표면과 다른 의도

## 3. 방어기제 (Defense Mechanisms)

### 대화에서 나타나는 방어기제

| 방어기제 | 정의 | 대화 예시 |
|----------|------|----------|
| **부정 (Denial)** | 불편한 현실 거부 | "그냥 친구야", "아무것도 아니야" |
| **투사 (Projection)** | 자기 감정을 상대에게 | "네가 날 싫어하잖아" (자신이 싫어함) |
| **합리화 (Rationalization)** | 그럴듯한 이유 대기 | "바빠서 연락 못 했어" (실제론 피함) |
| **회피 (Avoidance)** | 화제 전환 | 중요한 얘기 → "아 배고프다" |
| **수동적 공격 (Passive-aggressive)** | 간접적 적대감 | "아 그래? 잘됐네~", 냉소적 ㅋ |
| **반동형성 (Reaction Formation)** | 반대로 행동 | 좋아하면서 차갑게 대함 |

## 4. 관계 역학 모델

### 권력 균형 (Power Balance)

**평가 지표**:
- 누가 대화를 시작하는가?
- 누가 대화를 끝내는가?
- 누가 더 많이 양보하는가?
- 누가 결정권을 가지는가?

**불균형 신호**:
- 한쪽만 계속 먼저 연락
- 한쪽만 계속 맞춰줌
- 일방적인 감정 노동

### 상호의존 모델 (Interdependence Model)

| 유형 | 설명 | 대화 특징 |
|------|------|----------|
| **상호의존** | 둘 다 서로 필요 | 균형잡힌 주고받기 |
| **일방의존** | 한쪽만 의존 | 한쪽의 과잉 연락/관심 |
| **반의존** | 의존 거부 | 도움 요청 회피, "괜찮아" |
| **공의존** | 병적 상호의존 | 과도한 감정 개입, 경계 없음 |

## 5. 트라우마 관련 모델

### Complex PTSD 관계 패턴

**배신 트라우마 (Betrayal Trauma)** 피해자의 대화 특징:
- 과잉경계 (hypervigilance)
- 테스트 행동 (상대 신뢰도 시험)
- 자기개방 후 후회/철회
- 친밀감에 대한 양가감정

### 반복 강박 (Repetition Compulsion)

과거의 상처 주는 관계를 무의식적으로 반복:
- 비슷한 유형의 사람에게 끌림
- 같은 패턴의 갈등 반복
- "이번엔 다를 거야" but 결과는 같음

## 6. 스키마 이론 (Schema Therapy)

### 관계에서 활성화되는 핵심 스키마

| 스키마 | 핵심 믿음 | 대화에서 나타남 |
|--------|----------|----------------|
| **유기/불안정** | "결국 떠날 거야" | 집착, 확인 요구, 불안 표현 |
| **불신/학대** | "날 이용할 거야" | 의심, 테스트, 방어적 |
| **결함/수치심** | "나는 부족해" | 자기비하, 과잉 사과 |
| **정서적 결핍** | "내 감정은 중요하지 않아" | 감정 표현 억제, 맞춤 |
| **복종** | "상대 원하는 대로 해야 해" | 의견 없음, 항상 동의 |
| **엄격한 기준** | "완벽해야 해" | 비판적, 통제적 |
| **특권의식** | "내가 우선" | 일방적 요구, 공감 부족 |

## 7. 분석 적용 가이드

### 단계별 적용

1. **1차 스캔**: 대화 전체에서 눈에 띄는 패턴 식별
2. **모델 매칭**: 어떤 심리학적 모델이 가장 적합한지 판단
3. **가설 수립**: "이 사람은 OO 때문에 XX 하는 것 같다"
4. **Q&A 검증**: 질문을 통해 가설 검증
5. **종합 해석**: 여러 모델을 통합한 해석

### 주의사항

- **진단하지 말 것**: 임상적 진단은 전문가 영역
- **라벨링 주의**: "너는 회피형이야"처럼 단정 금지
- **맥락 중시**: 상황에 따라 누구나 다른 패턴 보임
- **유연성 유지**: 모델은 도구일 뿐, 맹신 금지
