# 카카오톡 대화 심층 분석 프레임워크

## 개요

카카오톡 대화를 단순 요약이 아닌 **심리학적/관계역학적 관점**에서 분석하고, 상담식 Q&A를 통해 숨겨진 의미와 패턴을 발견하는 프레임워크입니다.

## 분석 카테고리

### 1. 로맨틱 관계 분석 (Romantic Analysis)
- 썸/호감 가능성 평가
- 관계 진전 신호 탐지
- 상대방 진심 여부 분석
- 관계 발전 전략 제안

### 2. 비즈니스 갈등 분석 (Business Conflict Analysis)
- 동료/파트너 간 숨겨진 갈등 요소
- 권력 역학 분석
- 의사소통 패턴의 문제점
- 갈등 예방 및 해결 전략

### 3. 친구 관계 위험신호 (Friendship Risk Analysis)
- 관계 악화 신호 탐지
- 뉘앙스 변화 감지
- 숨겨진 불만/서운함 파악
- 관계 회복 전략

## 폴더 구조

```
analysis/
├── README.md                 # 이 파일
├── framework/                # 분석 방법론
│   ├── methodology.md        # 핵심 분석 방법론
│   ├── question-bank.md      # 상담식 질문 템플릿
│   └── psychological-models.md   # 심리학적 분석 모델
├── categories/               # 카테고리별 분석 가이드
│   ├── romantic.md           # 로맨틱 관계 분석
│   ├── business-conflict.md  # 비즈니스 갈등 분석
│   └── friendship-risk.md    # 친구 관계 위험신호
└── outputs/                  # 분석 결과 저장 (gitignore)
```

## 사용 방법

### 1. 대화 파일 준비
카카오톡 대화 내보내기 → `raw/` 폴더에 저장

### 2. 분석 실행
```
/analyze-kakao [파일경로] --category [romantic|business|friendship]
```

### 3. 상담식 Q&A
분석 후 Claude가 상담사 역할로 추가 질문을 통해 심층 분석 진행

### 4. 최종 보고서
분석 완료 후 종합 보고서 생성 → `outputs/` 폴더에 저장

## 분석 원칙

1. **판단 없이 관찰** - 도덕적 판단 배제, 객관적 분석
2. **패턴 인식** - 반복되는 행동/언어 패턴 추출
3. **맥락 중시** - 단일 메시지가 아닌 전체 맥락에서 해석
4. **다층 분석** - 표면적 의미 + 심층적 의미 + 관계 역학
5. **실용적 제안** - 분석에 그치지 않고 실행 가능한 제안 제공

## 프라이버시 주의

- 분석 결과는 `outputs/` 폴더에 저장되며 `.gitignore`에 포함
- 민감한 개인정보는 분석 보고서에서 익명화 처리
- 제3자 공유 금지
